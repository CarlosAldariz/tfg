El uso e iniciación de este proyecto es muy sencillo: 

	añade una base de datos, eliminale el nombre y la contraseña, pues ha de ser libre para el uso de esta 
	
	aplicación, a la base de datos que te damos llamala surfdamorte	

	Descarga el zip en tu ordenador 
	
	Descomprimelo sin cambiar nada en un servidor//servidor local 

	Entra en la carpeta tfg-main desde el servidor y sebería arrancarse automáticamente 

	ejemplo de path en local: localhost/tfg-main/

	En caso de querer cambiar este path tendrás que alterar el codigo de 
	
	/tfg-main/admin/config
